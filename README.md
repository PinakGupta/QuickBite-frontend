# QuickBite 🍔🚀

## 🌐 [LIVE DEMO](https://quickbite-frontend-1.onrender.com/) 🍽️

Welcome to **QuickBite**, the ultimate food delivery platform that connects food lovers with their favorite restaurants seamlessly! Browse menus, place orders, and enjoy delicious food delivered right to your doorstep - all through an elegant and intuitive interface.

---

## 🚀 Introduction

**QuickBite** is a modern food delivery platform designed to bridge the gap between hungry customers and local restaurants. With a user-friendly interface and powerful features, QuickBite makes ordering food online a breeze. Whether you're craving pizza, sushi, or a burger, our platform helps you discover and order from restaurants near you with just a few clicks.

For the **backend repository**, please visit: [QuickBite Backend](https://github.com/PinakGupta/QuickBite-backend)

---

## ✨ Features

- **Responsive Design** 📱: Fully optimized for all devices - mobile, tablet, and desktop
- **Restaurant Integration** 🏢: Restaurants can register, manage their profile, and showcase their menu
- **User Accounts** 👤: Simple registration process with personalized user profiles
- **Comprehensive Menu Browsing** 📋: Explore full restaurant menus with detailed item descriptions
- **Smart Search & Filters** 🔍: Find restaurants by cuisine type, location, and more
- **Secure Payments** 💳: End-to-end payment system powered by Stripe
- **Order Tracking** 📦: Real-time updates on order status
- **Restaurant Dashboard** 📊: Dedicated interface for restaurants to manage orders
- **Intuitive Cart Management** 🛒: Easy-to-use cart with modification options
- **Authentication System** 🔒: Secure login and registration flow with Auth0

## 🛠️ Tech Stack

- **Frontend**:
  - [React.js](https://reactjs.org/) ⚛️: For building dynamic user interfaces
  - [Redux](https://redux.js.org/) 🗂️: For efficient state management
  - [Material-UI](https://mui.com/) 🎨: For sleek, responsive components

- **Backend**:
  - [Node.js](https://nodejs.org/) 🟢: Server-side JavaScript runtime
  - [Express.js](https://expressjs.com/) 🚀: Fast, minimalist web framework
  - [MongoDB](https://www.mongodb.com/) 🍃: NoSQL database for flexible data storage

- **Authentication & Security**:
  - [Auth0](https://auth0.com/) 🔒: For robust authentication services
  - [JWT](https://jwt.io/) 🎫: For secure token-based authentication

- **Payment Processing**:
  - [Stripe API](https://stripe.com/) 💳: For secure, reliable payment processing

- **Media Management**:
  - [Cloudinary](https://cloudinary.com/) ☁️: For optimized image storage and delivery

- **Deployment**:
  - [Render](https://render.com/) 🌐: For web service hosting

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or later)
- npm or yarn
- MongoDB (local or cloud instance)

### Installation

1. Clone this repository:
   ```bash
   git clone https://github.com/PinakGupta/QuickBite-frontend.git
   ```

2. Navigate to the project directory:
   ```bash
   cd QuickBite-frontend
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Create a `.env` file in the root directory and add the necessary environment variables:
   ```
   REACT_APP_API_URL=your_backend_api_url
   REACT_APP_STRIPE_PUBLIC_KEY=your_stripe_public_key
   REACT_APP_AUTH0_DOMAIN=your_auth0_domain
   REACT_APP_AUTH0_CLIENT_ID=your_auth0_client_id
   ```

5. Start the development server:
   ```bash
   npm start
   ```
   The app will be available at [http://localhost:3000](http://localhost:3000).
